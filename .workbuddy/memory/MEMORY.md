# MEMORY.md - 长期记忆

## 项目：沿海大区运营周报生成

### 数据口径（核心规则，勿忘）

**台账筛选规则：**
- 台账按「年份(意向日期)+周数字段」筛选，不按日期范围（因多年数据共享同一周数）
- 大豆不做价格过滤，所有国产大豆记录均计入统计

**品种映射：**
| 周报名称 | 台账「品种」字段 |
|---------|--------------|
| 内贸玉米 | 国产玉米 |
| 进口DDGS | DDGS |
| 食用小麦 | 食用小麦 |
| 饲用小麦 | 饲用小麦 |
| 大豆 | 国产大豆（不过滤价格，所有记录均计入） |

**周期定义：** 每周六 ~ 下周五

### 用户要求
- 不要没问清楚需求就开始开发，先确认数据口径

- 格式参考：「本周意向销售8.83万吨（环比减少0.04万吨）...」
